1. Скачиваем архив с проектом из репозитория на github
2. Открываем через OpenServer. Для этого копируем из архива папку test-task-master в папку domains (в OSPanel). Далее запускаем OpenServer и в настройках во вкладке Домены добавляем домен, выбрав путь: test-task-master/public
3. Импортируем базу данных из папки data
4. Запускаем проект
